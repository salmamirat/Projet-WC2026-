# Projet-WC2026 - API REST

## Technologies
- Node.js
- Express.js
- PostgreSQL
- Sequelize

## Installation

```bash
npm install
npm start
```

## Endpoints

### Arbitres
- GET /arbitres
- GET /arbitres/:id
- POST /arbitres
- PUT /arbitres/:id
- DELETE /arbitres/:id

### Matchs
- GET /matchs
- GET /matchs/:id
- POST /matchs
- PUT /matchs/:id
- DELETE /matchs/:id

### Affectations
- GET /affectations
- POST /affectations
